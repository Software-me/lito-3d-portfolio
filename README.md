#  Interactive 3D Developer Portfolio

This project is a modern, interactive 3D developer portfolio built with React, Vite, React Three Fiber, and Tailwind CSS. It showcases my projects, technical skills, and background through a visually engaging and performance-focused web application.

The application combines real-time 3D rendering with a responsive single-page layout, allowing users to navigate smoothly between sections while experiencing an immersive interface.

##  Architecture Overview

The application follows a modular and scalable structure:

- **Components:** Reusable UI sections for maintainability
- **3D Scene Layer:** Managed through React Three Fiber for rendering and interaction
- **Styling:** Utility-first design using Tailwind CSS
- **Routing/Navigation:** Smooth scrolling and section-based navigation

---

##  Performance Considerations

- Controlled rendering to reduce unnecessary re-renders
- Optimized 3D assets for web performance
- Lightweight component structure for faster load times
